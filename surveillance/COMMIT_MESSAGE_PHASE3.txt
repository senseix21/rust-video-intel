feat: Complete Phase 3 - ROI Zone Detection Integration

Implement real-time zone detection filtering and UI integration for 
the surveillance system. Detections are now automatically assigned to 
zones and displayed in the TUI.

## Changes

### Core Functionality (app.rs)
- Add `get_detection_zone_name()` method to App
  - Returns zone name for detections inside enabled zones
  - Uses center-point detection method from Phase 1
  - Returns None if outside all zones or only in disabled zones

### UI Enhancements (ui.rs)
- Add "Zone" column to detection table (now 6 columns)
  - Shows zone name or "-" if outside zones
  - Updates automatically every frame
  
- Add zone summary panel to left sidebar
  - New 4th panel showing ROI zones overview
  - Displays up to 5 zones with status and counts
  - Color-coded: ✓ (green enabled) / ✗ (red disabled)
  - Shows enabled/total ratio in title
  - Empty state handling
  
- Add zone field to selected detection details
  - Shows which zone contains selected object
  - Only displayed when detection is in a zone

- Adjust left panel layout proportions
  - Performance: 40% → 35%
  - Classes: 30% → 25%
  - Living Beings: 30% → 20%
  - ROI Zones: 0% → 20% (new)
